Add contact — send invite via Messages

This page mirrors the app flow: do not call a Cloud Function to send SMS. After the contact exists in Firestore with inviteStatus: 'pending', open the native SMS composer with a prefilled body.

The invite link matches the app: https://verify.crateluv.com/sms-compliance.html?contactId=<id> (consent + OTP; Telnyx sends the code).

Create the contact in your app first (or via console) so this ID and inviteStatus are set.
